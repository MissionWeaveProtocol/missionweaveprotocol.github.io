Dieses Repository enthält die offizielle Website und Dokumentation von
MissionWeaveProtocol.

- Website: <https://missionweaveprotocol.github.io/>
- Protokoll: <https://github.com/missionweaveprotocol/missionweaveprotocol>
- Python SDK: <https://github.com/missionweaveprotocol/python-sdk>

## Lokale Entwicklung

Verwende Node.js 22 oder neuer.

```bash
npm ci
npm run docs:dev
```

Führe vor dem Öffnen eines Pull Request die vollständige lokale Prüfung aus:

```bash
npm run check
```
