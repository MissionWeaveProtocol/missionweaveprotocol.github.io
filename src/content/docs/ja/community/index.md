MissionWeaveProtocol はオープンに開発されています。変更を提案する前に、Organization 全体に適用される次のガイドラインを確認してください。

- [コントリビューションガイド](https://github.com/missionweaveprotocol/.github/blob/main/CONTRIBUTING.md)
- [行動規範](https://github.com/missionweaveprotocol/.github/blob/main/CODE_OF_CONDUCT.md)
- [セキュリティポリシー](https://github.com/missionweaveprotocol/.github/blob/main/SECURITY.md)

プロトコルのセマンティクス、JSON Schema、適合性に関する作業は
[プロトコルリポジトリ](https://github.com/missionweaveprotocol/missionweaveprotocol)
で行います。実装に関する作業は対応する公式 SDK リポジトリで行います。
[SDK の概要](../sdk/)から Python、Go、TypeScript、Java、Rust、C++ を選択してください。

## 脆弱性を報告する

セキュリティ脆弱性を公開 issue や pull
request で開示しないでください。再現手順、影響、推奨する緩和策を添えて、
[非公開の脆弱性レポートを送信してください](https://github.com/missionweaveprotocol/.github/security/advisories/new)。
