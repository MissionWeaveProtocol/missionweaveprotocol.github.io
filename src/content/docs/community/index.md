MissionWeaveProtocol is developed in the open. Before proposing a change, read
the organization-wide guidance:

- [Contribution guide](https://github.com/missionweaveprotocol/.github/blob/main/CONTRIBUTING.md)
- [Code of conduct](https://github.com/missionweaveprotocol/.github/blob/main/CODE_OF_CONDUCT.md)
- [Security policy](https://github.com/missionweaveprotocol/.github/blob/main/SECURITY.md)

Protocol semantics, schemas, and conformance work belong in the
[protocol repository](https://github.com/missionweaveprotocol/missionweaveprotocol).
Python implementation work belongs in the
[Python SDK repository](https://github.com/missionweaveprotocol/python-sdk).

## Report a vulnerability

Do not disclose security vulnerabilities in a public issue or pull request.
[Submit a private vulnerability report](https://github.com/missionweaveprotocol/.github/security/advisories/new)
with reproduction steps, impact, and any suggested mitigation.
