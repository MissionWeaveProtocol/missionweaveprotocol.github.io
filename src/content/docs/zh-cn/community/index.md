MissionWeaveProtocol 采用开放方式开发。提出变更前，请先阅读适用于整个 Organization 的指南：

- [贡献指南](https://github.com/missionweaveprotocol/.github/blob/main/CONTRIBUTING.md)
- [行为准则](https://github.com/missionweaveprotocol/.github/blob/main/CODE_OF_CONDUCT.md)
- [安全策略](https://github.com/missionweaveprotocol/.github/blob/main/SECURITY.md)

协议语义、JSON Schema 和符合性相关工作应提交到
[协议仓库](https://github.com/missionweaveprotocol/missionweaveprotocol)。实现相关工作应提交到对应的官方 SDK 仓库；请通过
[SDK 概览](../sdk/)选择 Python、Go、TypeScript、Java、Rust 或 C++。

## 报告安全漏洞

请勿在公开 issue 或 pull request 中披露安全漏洞。请通过
[私密漏洞报告](https://github.com/missionweaveprotocol/.github/security/advisories/new)
提交复现步骤、影响以及建议的缓解措施（如有）。
