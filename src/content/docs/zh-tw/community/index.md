MissionWeaveProtocol 採用開放方式開發。提出變更前，請先閱讀適用於整個 Organization 的指南：

- [貢獻指南](https://github.com/missionweaveprotocol/.github/blob/main/CONTRIBUTING.md)
- [行為準則](https://github.com/missionweaveprotocol/.github/blob/main/CODE_OF_CONDUCT.md)
- [安全策略](https://github.com/missionweaveprotocol/.github/blob/main/SECURITY.md)

協定語義、JSON Schema 和符合性相關工作應提交到
[協定儲存庫](https://github.com/missionweaveprotocol/missionweaveprotocol)。實作相關工作應提交到對應的官方 SDK 儲存庫；請透過
[SDK 概覽](../sdk/)選擇 Python、Go、TypeScript、Java、Rust 或 C++。

## 報告安全漏洞

請勿在公開 issue 或 pull request 中披露安全漏洞。請透過
[私密漏洞報告](https://github.com/missionweaveprotocol/.github/security/advisories/new)
提交復現步驟、影響以及建議的緩解措施（如有）。
